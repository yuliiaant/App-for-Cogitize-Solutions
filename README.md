## The challenge

User should be able to:

- Switching between tabs
- Add new card to the list
- Change card props and name
- Drag and drop to reorder cards on the list

## Key fecnologies

- React.js
- Next.js
- Redux/Redux-Toolkit
- TypeScript
- LocalStorage
- react-beautiful-dnd
- SCSS/BEM
- deploy: Vercel
